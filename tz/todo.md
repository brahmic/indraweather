# TODO: анимация спутниковых кадров

## Цель

Дополнить обычный актуальный снимок Кемь - Кандалакша анимацией движения
облачности. Анимация строится из всех успешно собранных ИК-кадров за скользящие
12 часов и не заменяет существующие снимки.

## Реализация

- [x] Выбрать единый ИК-слой: он доступен днём и ночью и не даёт визуального
  скачка между True Colour и ночным каналом.
- [x] Выделить независимую долговечную очередь сбора с интервалом 20 минут.
- [x] Хранить PNG в отдельном Docker volume, а метаданные и статусы задач - в
  PostgreSQL.
- [x] Удалять кадры и результаты старше 26 часов.
- [x] Собирать короткий MP4 из кадров не старше 12 часов; это пользовательская
  анимация, а не буквальный GIF, и она заметно компактнее.
- [x] Добавить нейтральный тип вложения `animation` и адаптеры Telegram/MAX.
- [x] Приложить анимацию к плановым бюллетеням и к `/weather`, не создавая её
  повторно для каждого получателя.
- [x] Отмечать в подписи фактический временной диапазон и число кадров.

## Результат после запуска

Сразу после развёртывания анимация появится после накопления минимум трёх
кадров. За первые 12 часов её охват растёт до 12 часов; пропуски EUMETSAT не
блокируют бюллетень и не исключают уже сохранённые кадры.
